# Validated Data

This directory contains validated results organized by domain.

## Structure
- `physics/`: Validated physics claims (CSVs, source links).
- `biology/`: Validated biology claims (PDB analysis results).
- `neuroscience/`: Validated neuroscience claims (if any).
- `social_science/`: Validated social science claims (if any).
- `mathematics/`: Validated mathematical derivations.
