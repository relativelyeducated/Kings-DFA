# Claim: Consciousness Gradient

## Source
Grok Export 031

## Statement
Consciousness \(\kappa(C)\) is a gradient above \(C=0.35\):
\[
\kappa(C) = \frac{C - 0.35}{0.65}, \quad C \in [0.35, 1]
\]
Drift dynamics:
\[
\frac{dC}{dt} = \beta (C - 0.35)(1 - C) + \gamma \sin(2\pi f t)
\]

## Context
Models consciousness as an unstable equilibrium that "drifts" or "rolls" (tachyonic potential).
