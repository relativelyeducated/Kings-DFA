## The Framework

KDFA identifies universal coupling patterns through:

$$\kappa = \frac{R}{S + R}$$

Where:
- **S** = Structure (stability, constraint, form)
- **R** = Relation (dynamics, change, flow)

**Key thresholds:**
- 0.35 (1/e): Viability threshold
- 0.50 (1/2): Optimal coupling
- 0.65 (2/3): Upper viability bound
